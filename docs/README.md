### 기능 목록 작성

## 세팅

- 오픈그래프와 트위터카드
- 폰트
- 아이콘

## 헤더 (다같이)

- 드롭다운 메뉴
  - tab
  - 각 tab에 해당하는 컨텐츠 출력
- 네비게이션
  - 가입(링크)
  - 검색 애니메이션

## 바디

### 현민

- 이미지 애니메이션
- 버튼
- 플로팅 이벤트 (애니메이션)

- 공지사항 (시간에 따라 데이터 변경)
  - 이벤트 페이지 슬라이드
- 커피 프로모션
- top 버튼

### 혜주

- 동영상 배경
- 스크롤위치 배경

### 다같이

- 다중요소 슬라이드
